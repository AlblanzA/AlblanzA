## Featured project: AstroTrader Pro

**Classical financial astrology with professional astronomical calculation.**

A privacy-first hybrid app combining a Python backend (Swiss Ephemeris, arcsecond accuracy) with a modern HTML/JS/Canvas frontend, packaged for Windows desktop and Android (Google Play, coming soon).

**Highlights:** 200+ financial assets mapped to ruling planets, interactive zodiac wheel, time projections with bull/bear ranking, retrograde-aware aspect duration up to 7 years, 13 native languages including RTL Arabic, RevenueCat subscription billing, privacy-first with zero trackers.

**Stack:** Python 3.10, pyswisseph, pywebview, bottle, HTML5, CSS3, Vanilla JavaScript, Capacitor 6, astronomy-engine, RevenueCat 9.2, Inno Setup 6, GitHub Pages.

Public landing page: https://alblanza.github.io/astrotrader-public/

## Why hybrid HTML + Python?

Same architectural choice as Discord, VS Code, Spotify, Slack, Notion: scientific computation in Python where precision matters, modern UI in HTML/JS where flexibility matters. Single codebase running natively on Windows desktop, Android mobile and the web.
